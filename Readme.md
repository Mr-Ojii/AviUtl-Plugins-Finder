# AviUtl-Plugins-Finder
exeが存在しているディレクトリ以下のフォルダを検索し、  
検索したフォルダ内に存在しているAviUtlのPluginの  
``拡張子・プラグイン情報・フルパス``を記載したcsvファイルを作成するプログラム

## How-To-Build
#### Using MinGW
Run ``Build-MinGW.cmd``

#### Using Visual Studio
Run ``Build-VS.cmd`` on ``Visual Studio Developer Command Prompt``
